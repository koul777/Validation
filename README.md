# CSV AI 문서 자동화 워크벤치

제약·바이오 컴퓨터 시스템 밸리데이션(CSV) 교육에서 사용할 수 있는 정적 웹앱입니다.

## 실행 방법

문서 자동 생성 기능만 사용할 경우 `index.html` 파일을 브라우저에서 열면 바로 실행됩니다.

OpenAI API 보강 기능까지 사용할 경우 로컬 서버로 실행하는 방식을 권장합니다.

```powershell
node server.js
```

브라우저에서 `http://127.0.0.1:8080`을 엽니다.

## 주요 기능

- 요구사항/업무 메모를 URS 초안으로 정규화
- 요구사항별 위험평가, RPN, 위험등급 자동 도출
- 위험 기반 IQ/OQ/PQ 테스트 초안 생성
- URS-RA-Test 추적성 매트릭스 생성
- 최종보고서 초안 및 AI 검토 프롬프트 생성
- OpenAI API 키 입력 후 AI 검토/보강 결과 생성
- Markdown 패키지와 CSV 매트릭스 다운로드

## 교육 활용 포인트

- 반복 문서 작성 업무 중 자동화 가능한 영역 식별
- AI가 생성한 초안을 SME/QA/CSV 담당자가 검토하는 방식 실습
- 기존 SOP와 템플릿 문구를 프롬프트 근거로 재사용하는 방법 실습
- 데이터 무결성, 감사추적, 전자서명, 접근권한 항목의 누락 점검

## 주의

이 앱은 교육용 규칙 기반 생성기입니다. 생성 결과는 승인 문서가 아니며 실제 프로젝트에서는 승인된 Validation Plan, SOP, 시스템 설계문서, 테스트 증거와 대조 검토해야 합니다.

OpenAI API 키는 코드에 저장하지 않고 브라우저 세션에만 보관합니다. 공용 PC, 배포 서버, 화면 공유 환경에서는 API 키 입력과 노출에 주의해야 합니다.
